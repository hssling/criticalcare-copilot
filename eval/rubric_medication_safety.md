# Medication-Safety Rubric

| Dim | 0 | 3 | 5 |
|---|---|---|---|
| Duplicate therapy | misses | catches common | catches subtle (e.g., two classes of PPI) |
| Allergy conflict | misses severe | catches most | catches with cross-reactivity notes |
| Renal/hepatic dosing | ignores | flags obvious | correct threshold citations |
| Monitoring completeness | none | partial | catches missed levels + intervals |
| Prophylaxis bundle | none | VTE or SUP alone | both + contraindication awareness |
| Phrasing safety | imperative orders | mostly advisory | fully advisory + escalation |
