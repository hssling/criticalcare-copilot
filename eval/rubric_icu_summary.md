# ICU Summary Rubric

Score each candidate summary 0-5 on each dimension. Mean across dimensions ≥ 4.0 is release-quality.

| Dim | 0 | 3 | 5 |
|---|---|---|---|
| Active problems completeness | misses critical problems | lists most | lists all, appropriately prioritized |
| Trend interpretation | ignores trends | notes some | correctly interprets and links to management |
| Red-flag detection | misses all | catches major | catches major + subtle |
| Missing-data identification | none | partial | comprehensive |
| Safety phrasing | contains imperative orders | mostly advisory | fully advisory, with escalation |
| Handoff usefulness | unusable | acceptable | immediately useful |

Scorers: ≥2 clinicians, with reconciliation on disagreements ≥1.
